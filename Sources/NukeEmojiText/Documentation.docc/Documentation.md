# ``NukeEmojiText``

Display text with custom emojis in the format `:emoji:`. Remote emojis are loaded and cached using [Nuke](https://github.com/kean/Nuke)

## Overview

Wrapper around `EmojiText` to use [Nuke](https://github.com/kean/Nuke) for loading and caching remote emojis.
